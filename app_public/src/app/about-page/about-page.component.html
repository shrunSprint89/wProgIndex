This project is an open-source web application that aims to provide a mind map with short notes about web technologies and various areas of web programming and also to provide a handpicked collection of good resources to learn about various areas in web programming.

Software developers, web developers who want to get a birds-eye view of web programming and also want links to great resources for learning may find this website useful. Anyone looking to find short notes for revising too.

PS: This project is in progress. Links to resources to be added shortly. Adding new features every week.

Code: https://github.com/shrunSprint89/wProgIndex/

Libraries used:

  • markmap - https://github.com/dundalek/markmap
  • particles.js - https://github.com/VincentGarreau/particles.js
